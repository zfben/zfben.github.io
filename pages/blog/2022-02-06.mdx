我们经常遇到不同的人，感觉自己跟他们不一样，但是说不清哪里不一样，常常是一句三观不同作为结论。

但今天看到[《happy letter》](http://xiao.do/issues/amor-fati-1011664)中分享了一张图，列举了三种人生观的取向：

|| 快乐的人生 | 有意义的人生 | 精神富足的人生 |
| - | - | - | - |
| 关键词 | 舒适、愉快、安全 | 显著，有目的性，持之以恒 | 多样化，兴趣导向，变化的观点 |
| 要素 | 金钱、时间、人际关系、积极态度 | 道德标准、一致性、人际关系、信仰 | 好奇心、时间、能量、自发性 |
| 结果 | 个人满足 | 社会贡献 | 智慧 |
| 指标 | 生活满意度、正面情绪 | 生活的意义、客观的意义 | 精神富足的生活和体验 |

这是一份社会科学研究报告中的一部分，科研的好处之一是会帮我们把模糊的问题给界定清晰，像这样把人生观区分为三种，就很容易让人理解并对号入座了。

虽然人是复杂的，100% 属于其中某一个的人很少，但通过把这三种取向视为三个向量，从而可以帮助我们意识到自己和他人人生观取向中哪个方向多一些，哪个方向少一些。

我的主要取向是精神富足，兴趣一直是我做事的主要驱动力，并且我也喜欢多样性（因为兴趣本身就是多变的），以至于作为程序员的我，大部分朋友都不是程序员，甚至不是互联网行业的从业者。通过跟他们的交流，极大的满足了我对多样性的好奇心。

同时，我也有一部分追求意义的人生观，这也是为什么在我公司的官网上，会有专门的**企业社会责任**一项。虽然去年年底才刚注册，但我希望这是一个重视社会责任的公司，而不纯粹是为了获取更多的收入。

追求快乐也是我的一部分，毕竟在条件允许的情况下追求舒适、愉悦和安全是人的本性。而且按我的经验，这点在照顾长辈时特别管用，虽然长辈们嘴上说要节约，但对于更好的产品或服务，身体是很诚实的。（特别是带长辈出去陌生的地方旅游时。）

最后也提醒一下，就像各种人格分析工具一样，仅靠三个向量是不足以代表一个人的人生观的，而且随着人生阅历的增长和现实境遇的变化，人生观也会不停的发生变化。所以没必要刻意把自己框在某种观念中，对于怎么过自己这一生，自己永远是最好的老师，社会是最好的学校。

你的人生观取向是什么呢？
